# Überblick über den KI-Assistenten {#ai-assistant}

Der KI-Assistent ist eine Funktion der Benutzeroberfläche, mit der Sie durch Adobe-Konzepte navigieren und diese verstehen können. Sie können den KI-Assistenten nutzen, um Anwendungsfälle zum Erwerb von Produktwissen in verschiedenen Produkten von Adobe Experience Cloud besser zu verstehen – inklusive der Funktion „Komposition föderierter Zielgruppen“.

>[!CAUTION]
>
>Bevor Sie den KI-Assistenten verwenden können, müssen Sie den Benutzerrichtlinien für generative KI in Adobe Experience Cloud zustimmen. Weitere Informationen zur Vereinbarung finden Sie im [Überblick über den KI-Assistenten (Vorgängerversion)](https://experienceleague.adobe.com/de/docs/experience-platform/ai-assistant/home){target="_blank"}.

Bei der Komposition föderierter Zielgruppen können Sie auf Produktwissen zugreifen, um mehr über Adobe-Konzepte im Zusammenhang mit verschiedenen Prozessaspekten zu erfahren. Der KI-Assistent unterstützt vier Arten von Anwendungsfällen: **Offene Erkennung** (erkunden Sie Produktkonzepte basierend auf der Experience League-Dokumentation), **Gezieltes Lernen und Fehlerbehebung** (stellen Sie Fragen zu bestimmten Funktionen), **Betriebliche Erkenntnisse** (stellen Sie Fragen zu Ihren Objekten in der Komposition föderierter Zielgruppen) und **Erstellen von Kompositionen föderierter Zielgruppen**.

## Zugriff {#access}

Greifen Sie auf den KI-Assistenten zu, indem Sie ![das entsprechende Symbol](/help/start/assets/ai-assistant/icon.png) in der oberen Leiste auswählen. Der KI-Assistent wird rechts am Bildschirm angezeigt. Sie können ![Alt-Text des Bildes anzeigen](assets/do-not-localize/Smock_FullScreen_18_N.svg "das Erweiterungssymbol") auswählen, um das Fenster des KI-Assistenten zu erweitern.

![Das Symbol für den KI-Assistenten ist hervorgehoben und zeigt an, wie auf den KI-Assistenten zugegriffen werden kann.](/help/start/assets/ai-assistant/access.png)

## Verwenden des KI-Assistenten {#using}

Geben Sie nach dem Öffnen des KI-Assistenten Ihre Frage in das Feld am unteren Bildschirmrand ein und drücken Sie die Eingabetaste. Die Antwort auf Ihre Frage wird jetzt angezeigt. Sie können „Daumen hoch“ oder „Daumen runter“ verwenden, um die Antwort zu bewerten.

![Im KI-Assistenten wird eine Beispielfrage und -antwort angezeigt.](/help/start/assets/ai-assistant/sample-question-answer.png)

## Beispielfragen {#sample-questions}

Die folgenden Abfragen veranschaulichen die verfügbaren Fragetypen, die Sie dem KI-Assistenten stellen können:

| Abfragetyp | Beispielfrage |
| ---------- | --------------- |
| Offene Erkennung | <ul><li>Was ist die Komposition föderierter Zielgruppen?</li></ul> |
| Gezieltes Lernen | <ul><li>Wie kann ich ein Konto für eine föderierte Snowflake-Datenbank konfigurieren?</li><li>Wie kann ich eine föderierte Komposition erstellen?</li></ul> |
| Betriebliche Erkenntnisse | <ul><li>Wie viele föderierten Datenbanken habe ich in meiner Sandbox?</li><li>Wie viele Schemata habe ich in den letzten 30 Tagen erstellt?</li></ul> |
| Erstellen einer Komposition föderierter Zielgruppen | <ul><li>Erstellen Sie eine föderierte Zielgruppe mit Kundinnen und Kunden, die in Großbritannien leben.</li></ul> |

Darüber hinaus können Sie den KI-Assistenten nutzen, um eigenständig eine Komposition föderierter Zielgruppen zu erstellen.

## Erstellen einer Zielgruppe {#create-audience}

Sie können den KI-Assistenten verwenden, um mithilfe von Prompts in natürlicher Sprache eine Komposition föderierter Zielgruppen zu erstellen. Wenn Sie mit dem KI-Assistenten eine Zielgruppe erstellen, erarbeitet der KI-Assistent einen auf Ihrem Prompt basierenden Plan und führt ihn in Ihrem Browser mithilfe KI-gestützter Automatisierung aus.

Wenn Sie dem KI-Assistenten beispielsweise sagen „Erstelle mithilfe des Schemas CUSTOMERS_Table eine föderierte Zielgruppe von Kundschaft mit Wohnsitz in Großbritannien“, legt der KI-Assistent den Plan für die Erstellung der Zielgruppe dar. Dazu gehören Schritte wie das Navigieren zur Seite „Föderierte Kompositionen“, die Art und Weise der Erstellung der Komposition durch den Agent und das Speichern der Zielgruppe, sobald der Vorgang abgeschlossen ist.

![Es wird eine Beispielfrage und -antwort angezeigt.](/help/start/assets/ai-assistant/ask-create.png)

Wenn der Plan korrekt aussieht, können Sie **[!UICONTROL Ausführen]** wählen, damit der Agent die Automatisierung durchlaufen kann. Der Agent führt im Browser eigenständig die Schritte zur Erstellung Ihrer angeforderten Komposition in der Benutzeroberfläche für die Komposition föderierter Zielgruppen durch. Wenn Sie die Automatisierung zu einem beliebigen Zeitpunkt stoppen möchten, wählen Sie **[!UICONTROL Stoppen]**.

![Der Plan wurde realisiert und der Agent führt den Plan eigenständig aus.](/help/start/assets/ai-assistant/execute-plan.png)

Derzeit unterstützt die Fähigkeit zur Zielgruppenerstellung die folgenden zusätzlichen Funktionen:

- Planung
   - Sie können föderierte Kompositionen erstellen, die nach einem wiederkehrenden Zeitplan ausgeführt werden. Zu den unterstützten Werten gehören **Einmal** und **Täglich**.
- Deduplizierung
   - Sie können die föderierten Dateneinträge während der Datenabstimmung deduplizieren
